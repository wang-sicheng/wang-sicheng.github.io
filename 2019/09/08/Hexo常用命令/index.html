Hexo常用命令

发表于 2019-09-08 更新于 2019-09-13 分类于 Hexo

创建Hexo工程

$ hexo init [folder]

创建一个文件夹，使用hexo命令初始化为hexo工程目录。

新建文章

$ hexo new [name]

会在/blog/source/_POST/下生成name.md文件。

生成静态文件

$ hexo generate / $ hexo g
选项：
    -d, —deploy    文件生成后立即部署网站
    -w, —watch    监视文件变动

将md文件解析成HTML文件，存放至/blog/public/目录下。

启动

$ hexo server / $ hexo s
选项：
-p, —port    重设端口
-s, —static   只使用静态文件
-l, —log        启动日记记录，使用覆盖记录格式

默认网址为：http://localhost:4000/。

部署

$ hexo deploy / $ hexo d
选项：
-g, —generate    部署之前生成静态文件

将文章部署至服务器。

清除

$ hexo clean

清除缓存文件（db.json）和已生成的静态文件
